### 阶段二: 信效度检验 (Day 3)

#### 任务2.1: 信度分析
**方法**: Cronbach's Alpha系数

```python
# 分析量表
- TTF量表信度
- SI量表信度
- PV量表信度
- RD量表信度
- RV量表信度

# 判断标准
- α > 0.8: 优秀
- 0.7 < α < 0.8: 良好
- 0.6 < α < 0.7: 可接受
```

#### 任务2.2: 效度分析
**方法**: 探索性因子分析 (EFA) + 验证性因子分析 (CFA)

```python
# EFA步骤
1. KMO检验 (KMO > 0.7)
2. Bartlett球形检验 (p < 0.05)
3. 因子提取 (主成分分析)
4. 因子旋转 (最大方差法)
5. 因子载荷检验 (载荷 > 0.5)

# CFA步骤
1. 模型构建
2. 拟合指标评估 (CFI > 0.9, RMSEA < 0.08)
3. 聚合效度 (AVE > 0.5)
4. 组合信度 (CR > 0.7)
```

#### 任务3.1: 文本预处理
**输入**: Q19_pain_points开放题

```python
# 预处理步骤

3. 去除停用词
4. 词性标注
```

#### 任务3.2: BERT嵌入 + KMeans主题建模
**方法**: sentence-transformers + KMeans聚类

```python
# BERT主题建模分析
1. 使用paraphrase-multilingual-MiniLM-L12-v2模型生成384维嵌入
2. 使用轮廓系数确定最优聚类数K
3. KMeans聚类进行主题发现
4. jieba分词提取各主题关键词
5. 主题解释与命名

# 输出结果
- 痛点文本主题分布（10个主题）
- 各主题核心关键词
- 主题分布柱状图
```

#### 任务3.3: BERT情感分析
**方法**: transformers情感分类模型

```python
# 分析内容
1. 使用lxyuan/distilbert-base-multilingual-cased-sentiments-student模型
2. 对痛点文本进行情感分类（正面/负面/中性）
3. 分析不同主题的情感分布

# 分析结果
- 情感分布饼图
- 主题-情感交叉分析
```

### 阶段四: RQ2因果分析 (Day 5)

#### 任务4.1: DEMATEL分析
**目的**: 识别影响因素及其因果关系

```python
# 分析步骤
1. 构建直接影响矩阵 (6×6)
2. 计算规范化矩阵
3. 计算综合影响矩阵
4. 确定原因因素与结果因素
5. 绘制因果四象限图

# 输入因素
- F1: 数据孤岛
- F2: AI融合困难
- F3: 系统兼容性
- F4: 实施成本高
- F5: 合规风险
- F6: 人才短缺
```

#### 任务4.2: ISM分析
**目的**: 构建因素层级结构

```python
# 分析步骤
1. 确定阈值θ
2. 构建可达矩阵
3. 计算可达集和先行集
4. 层次划分
5. 绘制递阶层次结构图
```

#### 任务4.3: SEM结构方程模型
**理论基础**: TTF模型 + UTAUT模型

```
研究假设:
H1: TTF对感知价值(PV)有正向影响
H2: 社会影响(SI)对感知价值(PV)有正向影响
H3: 感知价值(PV)对使用意愿有正向影响
H4: 风险感知(RD)对使用意愿有负向影响
H5: SI在TTF与PV之间起调节作用
```

```python
# SEM分析步骤
1. 测量模型检验
2. 结构模型估计
3. 路径系数检验
4. 模型拟合评估
5. 假设检验结果

# 拟合指标标准
- χ²/df < 3
- CFI > 0.9
- TLI > 0.9
- RMSEA < 0.08
- SRMR < 0.08
```

## 方法选择指南

### 根据研究问题选择方法

| 研究问题 | 推荐方法 | 备选方法 |
|----------|----------|----------|
| 影响因素分析 | SEM | 回归分析、DEMATEL |
| 技术评估 | 模糊AHP-TOPSIS | DEA、熵权法 |

| 因素关系 | DEMATEL-ISM | ISM、解释结构模型 |

### 根据数据类型选择方法

| 数据类型 | 推荐方法 |
|----------|----------|
| 量表数据 | SEM、CFA、EFA |
| 分类数据 | 卡方检验、Logistic回归 |
| 文本数据 | BERT情感分析、jieba分词 |
| 时间序列 | 时间线分析、情景分析 |
| 模糊评价 | 模糊AHP、模糊TOPSIS |
