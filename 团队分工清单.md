## 六、辅助组灵活调配机制

### 6.1 调配原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         辅助组调配机制                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  【触发条件】                                                                │
│                                                                             │
│  1. 陶婉玉数据分析任务繁重，无精力做可视化                                    │
│  2. 绘图组进度滞后，需要支援                                                 │
│  3. 数据收集任务紧急，需要支援                                               │
│                                                                             │
│  【调配流程】                                                                │
│                                                                             │
│  张坦（监工）评估 ────▶ 确定支援需求 ────▶ 分配辅助组任务                    │
│       │                    │                    │                           │
│       │                    │                    │                           │
│       └────────────────────┴────────────────────┘                           │
│                            │                                                │
│                            ▼                                                │
│                    辅助组任务分配                                            │
│                    ├── 杨津墨：可视化支援/数据收集                           │
│                    └── 王小乐：可视化支援/数据收集                           │
│                                                                             │
│  【任务类型】                                                                │
│                                                                             │
│  ├── 可视化支援：协助绘图组制作图表                                          │
│  ├── 数据收集：问卷发放、访谈记录、案例收集                                  │
│  └── 文档整理：协助文字组整理文档                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 6.2 辅助组任务优先级

| 优先级 | 任务类型 | 说明 |
|--------|----------|------|
| 1 | 可视化支援 | 当陶婉玉无精力做可视化时优先支援 |
| 2 | 数据收集 | 问卷发放、访谈记录等 |
| 3 | 文档整理 | 协助文字组整理文档 |
