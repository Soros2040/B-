**生成时间**: 2026-02-21  
**分析工具**: Python 3.12 + BERT + KMeans + jieba  
**数据来源**: survey_data_simulated.csv (500份问卷)

---

## 一、分析方法说明

### 1.1 BERT主题建模

本研究采用BERT模型进行文本主题建模，具体方法如下：

**文本嵌入生成**：使用`paraphrase-multilingual-MiniLM-L12-v2`模型（384维）生成中文文本的语义嵌入向量。该模型支持多语言，能够捕捉中文文本的深层语义信息。

**主题聚类**：使用KMeans聚类算法对BERT嵌入向量进行聚类，通过轮廓系数确定最优聚类数（痛点文本K=10，需求文本K=10）。

**关键词提取**：使用jieba分词工具对每个主题的文本进行分词，统计词频提取关键词。

### 1.2 BERT情感分析

使用`lxyuan/distilbert-base-multilingual-cased-sentiments-student`模型进行情感分析，该模型支持多语言情感分类（正面/负面/中性）。

### 3.3 主题语义解读

## 六、方法对比

### 6.1 与传统方法对比

| 方法 | 传统方法 | BERT方法 |
|------|----------|----------|
| 主题发现 | 关键词字典匹配 | BERT嵌入 + KMeans聚类 |
| 语义理解 | 表层词汇匹配 | 深层语义向量表示 |
| 情感分析 | 情感词典计数 | BERT情感分类模型 |
| 准确性 | 中等 | 高 |
| 可解释性 | 高 | 中等 |

### 6.2 BERT方法优势

1. **深层语义理解**：BERT能够捕捉文本的深层语义信息，而非简单的词汇匹配。

2. **上下文感知**：BERT考虑词语的上下文环境，能够区分同一词语在不同语境下的含义。

3. **多语言支持**：使用多语言BERT模型，支持中文文本分析。

4. **端到端学习**：从原始文本到最终结果，无需人工设计特征。
