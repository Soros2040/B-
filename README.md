## 项目简介

本项目为"正大杯"全国大学生市场调查分析大赛参赛作品，聚焦于**Agentic RPA（智能代理自动化）**技术在金融经济领域的应用现状与市场需求调研。研究采用多方法三角验证策略，系统识别金融领域Agentic RPA应用的核心痛点、因果关系、需求优先级，并预测3-5年技术发展趋势。

## 研究目标

| 目标维度 | 具体内容 |
|----------|----------|
| 识别核心痛点 | 系统梳理金融领域Agentic RPA应用的核心痛点及其因果关系 |
| 挖掘潜在需求 | 从非结构化文本中提取隐含的技术需求和应用场景 |
| 评估技术路径 | 构建多维度技术评估模型，识别最优技术演进路径 |
| 预测发展趋势 | 基于现状分析，预测3-5年技术发展趋势（含AI+RPA融合趋势） |

## 七大端生态格局

根据"金融经济领域RPA需求与技术.xlsx"数据（168条记录），金融行业Agentic RPA应用形成七大端生态：

| 端 | 对应行业 | 主要角色 |
|------|----------|----------|
| **应用端** | 银行业、证券业、保险业 | 技术使用者 |
| **供应端** | RPA/Agent厂商 | 产品提供者 |
| **支撑端** | AI技术/金融科技商、基础设施/云厂商 | 技术支撑 |
| **治理端** | 监管机构/审计部 | 合规监管 |
| **交付端** | 系统集成商/咨询商 | 项目交付 |
| **环境端** | 垂直SaaS商 | 场景提供 |
| **人才端** | 高校/科研院所、人力资源 | 人才培养 |

## 核心研究发现

### SEM假设检验结果

| 假设 | 路径 | 系数(β) | 结论 |
|------|------|---------|------|
| H1 | TTF → PV | 0.270** | 支持 |
| H2 | SI → PV | 0.126** | 支持 |
| H3 | PV → BI | 0.219** | 支持 |
| H4 | RD → BI | -0.098** | 支持 |
| H5 | TTF×SI → PV | -0.066 | 不支持 |

### DEMATEL-ISM因果关系

- **原因因素**：数据孤岛、AI融合困难、系统兼容性
- **结果因素**：实施成本高、合规风险、人才短缺

### TOPSIS技术优先级

| 排名 | 技术方案 | 得分 |
|------|----------|------|
| 1 | LLM+RAG | 0.6448 |
| 2 | 低代码平台 | 0.5722 |
| 3 | 多模态AI | 0.5371 |
